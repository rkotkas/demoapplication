Welcome!

  • This is a Spring Boot application
  • There are 3 users in the database (username and password)
  • You have to log in with one of the user credentials (user registration is not implemented)
  • You are displayed a list of clients that you have entered. You are not allowed to see other users' clients.
  • You can add and edit clients
  • When a new client is saved, all mandatory fields are validated
  • All input data is stored to database
  • Country selection box values are populated from database.
